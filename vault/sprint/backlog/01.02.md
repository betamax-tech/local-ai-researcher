## Objective
Configure TypeScript compiler and build scripts to output both CommonJS and ESM artifacts.

## Acceptance Criteria
- [ ] `tsconfig.json` exists with `strict: true`, `noImplicitAny: true`, `noUncheckedIndexedAccess: true`.
- [ ] `lib` includes `ES2020` or later.
- [ ] `module` set to `ESNext` (dual-output handled by build script).
- [ ] `declaration: true` for `.d.ts` generation.
- [ ] `sourceMap: true` for debugging.
- [ ] Build script (esbuild or tsc with post-processing) produces:
  - `dist/index.cjs` (CommonJS)
  - `dist/index.mjs` (ESM)
  - `dist/index.d.ts` (TypeScript declarations)
- [ ] `npm run build` completes in < 5s.
- [ ] No TypeScript errors in source files.

## Architecture Notes

**Build strategy:** Use esbuild or tsc + post-processing to handle dual outputs. CommonJS should use `.cjs` extension; ESM should use `.mjs`.

All source TypeScript in `src/`, all output in `dist/`.
