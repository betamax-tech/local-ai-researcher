## Objective
Configure Jest test runner with TypeScript support and seed test utilities.

## Acceptance Criteria
- [ ] `jest.config.js` exists and uses `ts-jest` preset.
- [ ] Test files use `.test.ts` suffix and colocate with source.
- [ ] `npm run test` discovers and runs all test files.
- [ ] `npm run test:watch` launches in watch mode.
- [ ] Basic test template passes (e.g., `sum(1, 2) === 3`).
- [ ] Coverage reporting enabled; baseline > 0% for seed modules.
- [ ] No TypeScript errors in test files.

## Architecture Notes

**Test structure:**
```

  foo.test.ts
```
