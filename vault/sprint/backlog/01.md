## Objective
Initialize the project directory structure, install core dependencies, and establish TypeScript, linting, and testing infrastructure.

## Acceptance Criteria
- [ ] `package.json` exists with all core dependencies (TypeScript, ESM/CJS duals).
- [ ] `tsconfig.json` configured with strict mode enabled.
- [ ] `npm run build` compiles all source to `dist/` without errors.
- [ ] `npm run lint` runs ESLint with zero warnings/errors on seed files.
- [ ] `npm run test` launches Jest and runs basic test template.
- [ ] `npm run test:watch` works for development.
- [ ] Git is initialized and `.gitignore` includes common Node/TypeScript patterns.

## Architecture Notes

The project uses:
- **TypeScript 5.x** in strict mode
- **CommonJS + ESM** dual exports (`.cjs` and `.mjs` entrypoints)
- **ESLint 9.x** with `@typescript-eslint` preset
- **Jest 29.x** with TypeScript support via `ts-jest`
- **src/` for source, `dist/` for compiled output
- **tests/` for test files colocated (`.test.ts` suffix)

No monorepo or workspaces in v1.
