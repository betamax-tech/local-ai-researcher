## Objective
Implement low-level stdio message I/O: line-based reading, JSON framing, and write buffering.

## Acceptance Criteria
- [ ] `src/mcp/io.ts` exports `StdioTransport` class.
- [ ] `read()` method consumes lines from stdin, parses JSON, returns typed objects.
- [ ] `write()` method writes JSON to stdout with newline delimiter.
- [ ] Both methods are async and properly error on malformed input.
- [ ] Request/response ID correlation implemented (idempotency).
- [ ] No partial message corruption on incomplete reads.

## Architecture Notes

**Message format:** One JSON object per line (JSONL).

```typescript
export interface JSONRPCRequest {
  jsonrpc: "2.0";
  id: string | number;
  method: string;
  params?: unknown;
}

export interface JSONRPCResponse {
  jsonrpc: "2.0";
  id: string | number;
  result?: unknown;
  error?: { code: number; message: string; data?: unknown };
}
```
