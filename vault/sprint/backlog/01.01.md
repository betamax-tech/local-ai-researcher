## Objective
Define `package.json` with exact dependency versions, scripts, and dual-export configuration for CommonJS and ESM.

## Acceptance Criteria
- [ ] `package.json` declares `"main"`, `"module"`, `"exports"` fields supporting both CJS and ESM.
- [ ] All dependencies pinned to minor version (e.g., `^5.2.0`); no `latest` or wildcards.
- [ ] Scripts defined: `build`, `lint`, `test`, `test:watch`, `type-check`.
- [ ] `"type": "module"` or equivalent ESM-first configuration chosen.
- [ ] Dev dependencies include: `typescript`, `@typescript-eslint/{parser,eslint-plugin}`, `jest`, `ts-jest`, `@types/jest`, `@types/node`.
- [ ] `npm install` completes without errors or peer dependency warnings.
- [ ] Lock file (`package-lock.json` or `yarn.lock`) committed.

## Architecture Notes

**Dual exports strategy:**
```json
"exports": {
  ".": {
    "import": "./dist/index.mjs",
    "require": "./dist/index.cjs"
  }
}
```

Use `.mjs`/`.cjs` extension convention for dual output.
