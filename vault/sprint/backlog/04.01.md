## Objective
Implement local file/directory search adapter so `search(query, { sources: ['local'] })` searches local codebase.

## Acceptance Criteria
- [ ] `src/sources/local.ts` exports `LocalSearchAdapter` class.
- [ ] Scans `cwd` (or specified directory) for matching files.
- [ ] Searches file names, contents, and directory structure.
- [ ] Returns `SearchResult` items for each match.
- [ ] Respects `.gitignore`.
- [ ] Sets `source: 'local'` on all results.
- [ ] Implements simple relevance scoring (name match > content match > directory match).
- [ ] Excerpt extraction (30 lines by default).
- [ ] Timeout enforced (default 5s).
- [ ] Throws `SearchError` if directory not accessible.
- [ ] Unit tests: file name match, content match, .gitignore respect, timeout.

## Architecture Notes

**Search strategy:** Simple grep on file contents + filesystem traversal. No indexing in v1 (full-scan OK for typical project size < 100MB).
