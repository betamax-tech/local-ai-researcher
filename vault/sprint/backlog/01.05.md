## Objective
Set up GitHub Actions workflow to build, lint, and test on every push/PR to main.

## Acceptance Criteria
- [ ] `.github/workflows/ci.yml` exists.
- [ ] Workflow triggers on: push to main, PRs to main.
- [ ] Pipeline runs: `npm run type-check`, `npm run lint`, `npm run test`, `npm run build`.
- [ ] All steps complete without error.
- [ ] Workflow can be manually triggered.
- [ ] Job timeout is 10 minutes.

## Architecture Notes

**Workflow gates:** Lint → Type Check → Test → Build (parallel type-check and lint OK).
