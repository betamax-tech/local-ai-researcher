## Objective
Integrate web search via Brave Search API with fallback to local cache if API unavailable.

## Acceptance Criteria
- [ ] `src/sources/web.ts` exports `WebSearchAdapter` class.
- [ ] Uses Brave Search API (via environment variable `BRAVE_SEARCH_API_KEY`).
- [ ] Falls back to local search index if API unavailable (graceful degradation).
- [ ] Queries Brave with timeout (default 5s).
- [ ] Maps Brave results to canonical `SearchResult` format.
- [ ] URL canonicalization applied to result URLs.
- [ ] Excerpt extraction (30 lines by default).
- [ ] Throws `SearchError` with `SEARCH_TIMEOUT` or `SEARCH_SOURCE_UNAVAILABLE` on failure.
- [ ] Unit tests: successful search, API timeout, missing API key, fallback activation.

## Architecture Notes

**Adapter pattern:** Web search is a pluggable source. Other sources (local, custom) follow same pattern.

**Fallback strategy:** If Brave API unavailable, log warning and skip (no results), or use local index if available.
