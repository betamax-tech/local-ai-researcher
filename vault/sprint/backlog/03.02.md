## Objective
Implement the `search()` tool with excerpt-first retrieval and URL canonicalization for deterministic dedup.

## Acceptance Criteria
- [ ] `src/tools/search.ts` exports async `searchTool(query, options)` function.
- [ ] Supports `sources: ['web', 'local', 'custom']` (default all).
- [ ] Returns array of `SearchResult` with deterministic `id` based on canonicalized URL.
- [ ] URL canonicalization rules applied:
  - [ ] Scheme and hostname lowercase.
  - [ ] Remove `www.` unless only subdomain.
  - [ ] Strip trailing slash.
  - [ ] Sort query params alphabetically.
  - [ ] Strip fragments.
- [ ] Excerpts default 30 lines; `fullText: true` returns full content.
- [ ] `relevance` score (0–1) included for each result.
- [ ] Default `limit: 5`; respects `options.limit`.
- [ ] Timeout enforced (default 5s per source, configurable).
- [ ] Throws `SearchError` with codes: `SEARCH_TIMEOUT`, `SEARCH_SOURCE_UNAVAILABLE`, `SEARCH_MALFORMED_QUERY`.
- [ ] Unit tests: basic search, URL canonicalization, timeout, source filtering.

## Architecture Notes

**Excerpt logic:** Return first 30 lines of content (approximately), mark as excerpt if truncated.

**Canonical URL:** Used for dedup ID generation. Hash format: `sha256(canonicalURL + query)`.
